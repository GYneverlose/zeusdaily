# Zeus Daily Review Log

## 核心教训

1. **新闻时效性是铁律**：日报只收录当天新闻，搜索关键词必须包含精确日期。非当日内容不可接受
2. **每格Dashboard独立搜索**：不可复用其他格的搜索结果，避免数据交叉污染
3. **Silver等贵金属周末/亚洲时段数据延迟**：需标注"数据暂缺·截至Mon DD"而非硬编虚假数据
4. **英文版tooltip不可遗漏**：中英双版 .term 标签必须对称
5. **区块链板块最低3条**：需覆盖ETF资金流、监管进展、链上数据三个维度
6. **马来西亚板块最低3条**：需覆盖政策/经济/社会三个维度，含 .msia-tag
7. **趣闻必须当天**：不可用往期趣闻填充
